## Development

### Dependency isolation

This project encapsulates all development dependencies in a devcontainer.
To enable compatibility with ad-hoc site adjustments on GitHub via Github Codespaces
the preconfigured IDE is VSCode with extensions helpful for Astro development.

### Agent configuration

This project provides an [AGENTS.md](https://agents.md/) file for optimize working with
the agent of your choice.
